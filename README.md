# Renaissance 🏰

**2D Tower Defense** prototype built with Unity 6000.  
Pixel-art style, wave-based gameplay, 3 tower types with upgrades.

> ⚠️ **Demo pre-release** — testing phase. Feedback welcome!

## Gameplay

- Place towers on **Ground** tiles to defend your base
- 3 tower types: **Archer**, **Cannon**, **Mage**
- Earn coins by killing enemies → spend on **upgrades**
- Survive waves and defeat all enemies to **win**
- Press **ESC** to pause

## Controls

| Key | Action |
|-----|--------|
| Left Click | Place / select tower |
| Right Click | Cancel placement |
| Escape | Pause |
